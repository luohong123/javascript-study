#### 题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。
#### 思路一：每个台阶都有两种跳法（跳1阶和跳2阶），再加上跳n阶

- 代码
```js
function jumpFloorII(number)
{
    // write code here
   let n = 1;
    while(--number){ // 条件为真时循环，number为0（隐式转换为false）时停止循环
        n*=2;
    }
    return n;
}
```
- 执行结果

运行时间：91ms

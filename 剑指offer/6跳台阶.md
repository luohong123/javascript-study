#### 题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。
#### 思路一：动态规划
- 代码
```js
function jumpFloor(number)
{
    // write code here
   let x1 = 1,x2=1;
    while(number--){
        x2+=x1;
        x1=x2-x1
    }
     return x1;
}
```
- 执行结果

运行时间：79ms
占用内存：8444k

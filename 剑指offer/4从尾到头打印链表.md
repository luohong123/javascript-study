#### 题目描述
输入一个链表，按链表值从尾到头的顺序返回一个ArrayList
#### 思路一：压栈
循环链表，直到链表为空时停止循环，声明一个数组，把链表的val值压入到栈中，unshift()表示添加到数组的头部
- 代码
```js
/*function ListNode(x){
    this.val = x;
    this.next = null;
}*/
function printListFromTailToHead(head) {
    var res = [];
    while(head){
         res.unshift(head.val); // 添加到数组头部
         head=head.next;
    }
   return res
}
module.exports = {
    printListFromTailToHead : printListFromTailToHead
};
```

Binary Search

1. Sorted(单调递增或者递减)
2. Bounded(存在上下界)
3. Accessible by index(能够通过索引访问)

# 二分查找代码

```python
    left, right = 0, len(array) - 1
    while left <= right:
        mid = (left + right) / 2
        if array[mid] == target:
            # find the target
            break or return result
        elif array[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
```

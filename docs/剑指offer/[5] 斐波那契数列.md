## 题目

大家都知道斐波那契数列，现在要求输入一个整数 n，请你输出斐波那契数列的第 n 项（从 0 开始，第 0 项为 0）。
n<=39

## 方法一：递归

不考虑，有大量的重复计算，会导致内存溢出,在牛客网在线编程不会通过，

```js
function Fibonacci(n) {
  // write code here
  // 0 1 1 2 3 5 8 13 21 24
  if (n <= 0) {
    return 0
  }
  if (n == 1) {
    return 1
  }
  return Fibonacci(n - 2) + Fibonacci(n - 1)
}
```
